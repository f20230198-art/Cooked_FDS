# Cooked_FDS — Foundation of Data Science (CS G320) Study Guides

Exam-focused study notes for FDS, built from lecture slides/screenshots.
Each guide uses the same style: definitions, tables, formulas (LaTeX),
ASCII diagrams, fully-worked examples, and a quick-revision cheat-sheet.

## Contents

| File | Chapters | Topics |
|---|---|---|
| [FDS_Chapter_1_and_2_Study_Guide.md](FDS_Chapter_1_and_2_Study_Guide.md) | 1 & 2 | Intro to Data Science; Probability, Bayesian & Frequentist |
| [FDS_Chapter_3_Study_Guide.md](FDS_Chapter_3_Study_Guide.md) | 3 | Probability Distributions (Bernoulli, Binomial, Gaussian, Beta, …) |
| [FDS_Chapter_4_Study_Guide.md](FDS_Chapter_4_Study_Guide.md) | 4 | Optimization (Gradient Descent, LP, Lagrange Multipliers) |

## How to use

- Open the `.md` files in any Markdown viewer (VS Code preview, or `Ctrl+Shift+V`).
- Each chapter ends with a **cheat-sheet** + **practice questions** — use these for last-minute revision.
- Worked examples are solved step-by-step; the *method* matters more than the final decimal.

## Notes

- Math in blurry slide solutions was re-derived using standard formulas, so a few
  numbers may differ slightly from the originals (the method is correct).
- New chapters are added as separate files following the same format.
